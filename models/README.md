# Models Directory

Place your trained model files here.

## Expected Files

| File | Description | Size |
|------|-------------|------|
| `deforestation_trained_data.h5` | Trained CNN model | ~5-50 MB |

## How to Get the Model

### Option 1: Train Yourself
```bash
python src/training/DeForestation_Training.py
```

### Option 2: Download Pre-trained
Download from releases and place here.

> ⚠️ **Note:** `.h5` files are gitignored due to size. Use Git LFS or release assets.
